# Knowledge Log: Block and Inline
**Project/Lesson:** TOP Foundations - Block and Inline
**Date:** 2025-12-29
**Stakeholder:** @StarrDevelops

## Key Concepts
* **Block:** `div`, `h1-h6`, `p`. Spans full width; forces a new line.
* **Inline:** `span`, `a`, `img`. Only takes content width; padding/margins behave inconsistently on the vertical axis.
* **Inline-Block:** Hybrid. Sits on the same line but respects all Box Model properties.

## Verified Insights
* Centering a block element requires a defined width and `margin: 0 auto;`.
* Buttons are specialized elements that behave like inline-blocks to allow for internal padding.
